# **CryptoKey**

**CryptoKey** - Инструмент генерации паролей, ключей шифрования, паролей и тд...

## **Как работает**

- Работает он используя библиотеку `cryptography, base64, base58, secrets` (Python)

- Что доступно, подробнее (Базовые функции):

| Технологии | Функции |
|------------|---------|
| `Fernet` | Генерация ключей, шифрование и расшифрование |
| `RSA`| Генерация публичных и приватных ключей, шифрование и расшифрование |
| `Base64` | Кодирование и декодирование |
| `Base58` | Кодирование и декодирование |
| `AES (128/256)` | Генерация ключей, шифрование и расшифрование |

- Активно, Автор проекта `Alexx-coder или alexx` обновляет и добавляет новые криптографические функции

- Если хотите получать новые версии: заглядывайте в `папку CryptoKey (Там папки с версиями и их документацией)`

- Цель: `Мощный криптографический инструмент для каждого!`
