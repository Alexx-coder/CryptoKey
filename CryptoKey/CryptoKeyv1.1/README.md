# CryptoKey

CryptoKey - Инстурмент генерации паролей, ключей шифрования, расшифровки, зашифровки и много чего интересного

## Что нового добавили в CryptoKey v1.1

### Функции

#### Генерация пароля содержащий сид-фразы

- Сначала потребуется скачать файл со словами (Ссылка в примечаниях)

- Затем вставить в переменную `WORDLIST` путь к файлу

- Готово. Вы можете генерировать пароли.

- Пароль состоит из `12 сид фраз`

- `Энтропия: 77,5 Бит` - Подобрать невозможно

### Доработка

#### Подфункция Расшифрования сообщения через AES

- В прошлой версии (v1.1) Расшифрование через AES не было (Автор случайно удалил этот фрагмент безвозратно)

## Примечания

- [Ссылка на файл от EFF ](https://www.eff.org/files/2016/07/18/eff_large_wordlist.txt). `Данный файл имеет 7776 слов`

- Перед тем, как попасть в меню, вам придется согласиться с Политикой Конфедициальности, если отклоните программа аварийно завершится
