# **CryptoKey**

**CryptoKey** - Инструмент генерации паролей, ключей шифрования, расшифровки, зашифровки и много чего интересного...

**Автор: Alexx-coder или alexx (GitHub)**

**Версия: v1.3**

## **Что нового добавили в CryptoKey v1.3**


### **Функции**

#### **Функция 14 `ECC`**

- Внутри меню - 3 функции: `Генерация ECC ключей, Подпись сообщения и Проверка подписи`

- Тут мы используем технологии: `ECC`, `Serialization`, `SECP256K1`, `Base64`, `SHA256`

- Про Генерацию ECC ключей подробно:

| Как работает (Этапы) | Значение Этапов | Технологии |
|----------------------|-----------------|------------|
| 1. Генерация ключей | Генерируются ключи (KEY) | `ECC (Elliptic Curve Cryptography)`, `SECP256K1` |
| 2. Сериализация | Превращение объекта в строку (PEM) | `Serialization` |
| 3. Вывод ключей | Выводим наши ключи: Приватный и Публичный (PEM) | Нет |

- Теперь как работает Подпись сообщения, подробнее:

| Как работает (Этапы) | Значение Этапов | Технологии |
|----------------------|-----------------|------------|
| 1. Ввод пользователя (Приватный ключ PEM и Сообщение) | Нужно для следующих этапов | Нет |
| 2. Загрузка ключа (Из PEM в KEY) | Загружает приватный ключ в формате в KEY | `Serialization` |
| 3. Создание подписи | Создает подпись | `ECC` , `ECDSA`, `SHA256` |
| 4. Кодирование подписи в Base64 | Кодирует подпись в Base64 | `Base64` |
| 5. Вывод подписи (Base64) | Выводит подпись в Base64 | Нет |

- Ну и под конец про Проверка подписи, подробнее:

| Как работает (Этапы) | Значение Этапов | Технологии |
|----------------------|-----------------|------------|
| 1. Ввод пользователя (Публичный ключ, Сообщение и Подпись в Base64) | Нужно для следующих этапов | Нет |
| 2. Загрузка ключа (Из PEM в KEY) | Загружает публичный ключ в формате в KEY | `Serialization` |
| 3. Декодирование Подписи (Из Base64) | Декодирует подписи из Base64 | `Base64` |
| 4. Проверка подписи | Проверяет подпись | `ECC`, `ECDSA`, `SHA256` |
| 5. Вывод | Выводит верна или неверна подпись | Нет |
